# 🚀 VendorX – Vendor Performance Analytics Dashboard

## 📌 Overview

VendorX is an end-to-end data analytics project focused on analyzing vendor performance, cost efficiency, and operational trends.
It combines exploratory data analysis with interactive visualizations to deliver actionable insights through a web-based dashboard.

---

## 🎯 Objectives

* Analyze vendor performance and efficiency
* Identify cost trends and anomalies
* Detect outliers and inconsistencies
* Build an interactive dashboard for data exploration

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Streamlit

---

```
VENDORX/
```

## ▶️ How to Run Locally

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/VENDORX.git
cd VENDORX
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the Streamlit app

```
streamlit run app.py
```
